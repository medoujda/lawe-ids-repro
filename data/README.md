# Datasets

All five datasets used in this paper are publicly available. This directory
should have the following layout after download:

```
data/
├── unsw-nb15/
│   ├── UNSW_NB15_training-set.csv
│   └── UNSW_NB15_testing-set.csv
├── nsl-kdd/
│   ├── KDDTrain+.txt
│   └── KDDTest+.txt
├── cicids2017/
│   └── cicids2017_sampled.csv      ← generated by subsample_cicids2017.py
├── ton-iot/
│   └── *.csv                       ← TON-IoT Network dataset CSVs
└── cicids2023/
    └── *.csv                       ← CICIoT2023 CSV files
```

## CIC-IDS2017 subsample

The raw CIC-IDS2017 data is ~6 GB. We use a reproducible 500 000-row
stratified subsample (seed=42). To generate it from the raw files:

```bash
python data/subsample_cicids2017.py \
    --raw_dir /path/to/raw/cicids2017 \
    --out_dir data/cicids2017
```

The resulting `cicids2017_sampled.csv` is picked up automatically by
`preprocessing.py`.

## DATA_DIR environment variable

By default, `config.py` resolves all dataset paths relative to `./data/`.
To use a different root:

```bash
DATA_DIR=/my/datasets python run_lawe.py --dataset unsw-nb15
```
